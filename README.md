# # Proyecto Final - Sistema de Automatización de Información para la Empresa ABC

## Descripción del Proyecto

Este proyecto tiene como objetivo desarrollar un sistema informático para la empresa ABC, con el fin de automatizar la gestión de información. El sistema permitirá llevar un control eficiente de los datos manejados por la empresa, cumpliendo con una serie de requisitos establecidos.

## Requisitos

- **Facilitador**: Concepción Ortiz
## Criterios del Proyecto

El sistema debe cumplir con los siguientes criterios:

1. **Elaboración del proyecto**: Documentación y planificación del desarrollo del sistema.
2. **Instalación del sistema informático**: Instrucciones para la instalación del software en el entorno del usuario.
3. **Entrada o login al sistema**: Proporcionar un sistema de autenticación para los usuarios.
4. **Diseño de formularios interactuantes**: Crear formularios que permitan la interacción del usuario con el sistema.
5. **Programación de botones**:
   - **Nuevo**: Para crear nuevos registros.
   - **Guardar**: Para almacenar los registros.
   - **Modificar**: Para editar registros existentes.
   - **Eliminar**: Para eliminar registros no deseados.
   - **Imprimir**: Para generar impresiones de los registros.
   - **Salir**: Para cerrar el sistema.
6. **Consulta incremental de los registros**: Permitir la búsqueda de registros mediante un sistema de consulta eficiente.
7. **Diseño del menú principal**: Crear un menú que enlace a los formularios y reportes del sistema.
8. **Diseño y programación de reportes**: Generar reportes a partir de los datos almacenados en el sistema.

## Instalación

### Requisitos del Sistema

Antes de instalar el sistema, asegúrate de que tu entorno cumpla con los siguientes requisitos:

- **Sistema Operativo**: Windows 10 o superior
- **Framework**: .NET Framework 4.7.2 o superior
- **Base de Datos**: SQL Server (local o en la nube)
- **Espacio en Disco**: Al menos 200 MB de espacio disponible

### Pasos para la Instalación

1. **Descargar el Software**:
   - Descarga el archivo del instalador desde el siguiente enlace: [enlace a tu repositorio o archivo de instalación].

2. **Ejecutar el Instalador**:
   - Haz doble clic en el archivo descargado para iniciar el proceso de instalación.

3. **Seguir las Instrucciones del Asistente de Instalación**:
   - Acepta los términos de la licencia.
   - Selecciona la carpeta de destino donde se instalará el sistema.
   - Haz clic en "Instalar" y espera a que finalice el proceso.

4. **Configuración de la Base de Datos**:
   - Abre la aplicación de configuración de la base de datos incluida en el instalador.
   - Ingresa las credenciales de tu servidor de base de datos y completa la configuración inicial.

5. **Iniciar el Sistema**:
   - Una vez completada la instalación, busca el acceso directo en tu escritorio y haz clic para abrir el sistema.

## Uso

### Acceso al Sistema

1. **Iniciar la Aplicación**:
   - Haz clic en el acceso directo en tu escritorio para abrir el sistema.

2. **Login**:
   - En la pantalla de inicio, ingresa tu nombre de usuario y contraseña.
   - Haz clic en "Entrar" para acceder al sistema.

### Navegación por el Sistema

- **Menú Principal**: Desde el menú principal, puedes acceder a las siguientes opciones:
  - **Nuevo**: Crea un nuevo registro.
  - **Consultar**: Accede a la lista de registros existentes y realiza búsquedas.
  - **Reportes**: Genera reportes basados en la información almacenada.
  
- **Uso de Formularios**:
  - Al seleccionar una opción, se abrirá el formulario correspondiente.
  - Completa los campos requeridos y utiliza los botones para realizar acciones (Guardar, Modificar, Eliminar, Imprimir).

### Ejemplo de Navegación

1. Para crear un nuevo registro, selecciona la opción "Nuevo" en el menú principal.
2. Completa los campos del formulario.
3. Haz clic en "Guardar" para almacenar la información.
